About me

Hi, my name is Cosimo, I am 26 years old, from Italy. I like science, gaming, music and of course programming. I'm currently working as an Unity Generalist, C# and Shader developer. I'm also an Unity Asset Store publisher.
More about my work:
  • Started to get interested in programming since I was in school, and started expermienting with Pascal, C, and later VB.Net.
  • Around 2010 I started to work on some unofficial tools for the game Metin2, one of them was public too:
    • Metin2 Remote Console Tool (Italian): It was a tool which provided a connection to your game server and allowed you to execute mysql/ftp/game commands (ie banning/kicking people, players recap, create/remove items, etc.) or execute custom commands or SQL queries.
    • Metin2 Map Editor (Italian): I did this as I was experimenting with D3D9 libraries. It basically was my own (very basic) engine, based on dx9 libraries. It was able to render the map data you passed to it, and edit its heightmaps, 3D objects, and monster spawns.
    • A lot of minor ones.
  • When I stopped working on those tools, I wanted to create a simple game. I started another VB.Net + DX9 project, but it was obviously a very complex thing to do, so I started to study/use C#, and learn Unity a little bit.
  • After a while (2014/15) I started to work for a startup company: Crimson Games. We were a very small team, working on a big MMORPG project, Heroes of Asgard, which then got stopped halfway in 2016. I was working on the client, including:
    • Client side scripts
    • Entities sync with the server : players, actions, skills, monsters, items, etc.
    • Movements and inputs
    • Animators for characters and monsters
    • UI : menus, inventory, skills, buffs, minimap system, etc.
    • Map creation : worked on 3 big maps. I designed them, created the terrain and vegetation, and then filled with assets our artist did. I also created a daytime and weather system for our maps, which was then sync'd with the server.
    • Some shaders for multiple effects: rain/snow reacting materials for environment and characters, custom water shader(not the one below), distortions, animated UI materials
    • Basic visual effects : particle systems, other various VFX (example)
  • I worked on Habitat, for IndieVault gamejam 2016. I did the controls/characters movements, and some graphics.
  • Starting from late 2016 I also started some other Unity projects as I was experimenting and learning new stuff, planning to release some of them on the Asset Store too:
    • BendinGrass - Asset Store - WebGL
      I liked shaders, so I wanted to try Compute Shaders for grass touch bend. While I was developing it, I noticed that it could have been a good package to release too. I later had to drop compute shaders for compatibility issues, and use regular vertex/fragment shaders instead. I worked on this a lot, it now provides its own rendering system, specialized in packing volumetric data on a texture 2D and supporting hundreds of grass-bending objects thanks to the GPU computing. It also offers an handful of other features, such as space partitioning, dynamic volumes, terrain heightmap based optimizations

      BendinGrass is currently my biggest asset published on the store.
    • BubbleR - Asset Store
      Shader that I created for fun and to learn. It is an high quality, parametric bubble shader with a lot of customizable parameters, useful to achieve multiple different results with little effort, as you don't need to paint textures. Supports color noise, vertex animation, thin-film effect. It's also compatible with reflection probes and particles systems.
    • HoloGrid - Asset Store
      Another straightforward shader that I needed to render some grids and holograms. Decided to publish it on the Asset Store later on. Supports flickering, spatial noise, scan-line effect, view angle fading, triplanar mapping (very useful especially for grids, either world or local space).
    • SnoopBoard
      This one is just a for-fun project that I created while experimenting with compute shaders. It is a very basic drawing software with the essential drawing functions (draw&erase, undo&redo, generate a Snoop Dogg, etc.).
    • Caravan Card Game
      Yet another for-fun project, this time is a card game: Caravan!
      Based on the popular card game inside Fallout:New Vegas (with some very minor rules changes).
    • AudioSync
      A plugin useful for various effects based on audio signals, for example automatically animating your character as he/she speaks. It performs an FFT on the audioclip samples and provides frequency bands and other data, whit a customizable output function, which you can then use for whatever you want.
    • FloWater
      Simple water shader, with just refraction and 1 direction flow. After some time I decided to improve it, adding dynamic waterflow and more customizable colors. Eventually I will add reflections and other stuff too.
    • Other minor asset store projects I was working on (not mature/documented enough yet):
      • A triplanar shader pack, with some variants, like heightmap blending, bumpmap, splat maps, etc.
      • A sign creator.
      • Highlight and indicator shaders.
  • 2019-2020 I worked on Duke it Out in the Arena as a programmer.
    Duke it Out in The Arena is a multiplayer party game, in which the players have to achieve different goals, by using objects in the arenas to interact with other players. Goals may differ depending on the selected game mode. I'm part of the team as an Unity/C# programmer. I work on client, server, gameplay, and implementation of the various assets.
  • Early 2020-2021 working on Summertime Madness - Steam page

    Summertime Madness is a puzzle game (soon to be released on Consoles) that I worked on, as part of DP Games. I did all of the scripting, most of the shaders, Unity general stuff and some Playmaker state machines. A lot of cool stuff went into this game, in order to achieve decent illusions.
    Two of my favourites are probably the Escher room illusion and the Endless twisted corridor

  • 2021 - Rat Simulator

    Side-project started with DP Games. Currently very early stage of developing.
    At DP Games, we are very passionate about rats rights. We want to raise awareness about the struggle of being a rodent in the modern world.
    You have to feel the struggle of being a pest, and not a pet, of being victim of ratism, of living in the poorest part of the neighborwoods.

    You play as a rat, in order to complete your mission: survive another day... The end goal here, is to achieve a pleasant art style, combined with satisfying controls, while giving the player the full rat experience!
