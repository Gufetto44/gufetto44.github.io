About me

Hi, my name is Cosimo, I am 23 years old, from Italy. I like science, gaming, music and of course programming. I'm currently an Unity C# and Shader developer. I'm currently planning to release some packages on the Unity Asset Store.
More about my work:
  • Started to get interested in programming since I was in school, and started expermienting with Pascal, C, and then VB.Net.
  • Around 2010 I started to work on some unofficial tools for the game Metin2, one of them was public too:
    • Metin2 Remote Console Tool (Italian): It was a tool which provided a connection to your game server and allowed you to execute mysql/ftp/game commands (ie banning/kicking people, players recap, create/remove items, etc.) or execute custom commands or SQL queries.
    • Metin2 Map Editor (Italian): I did this as I was experimenting with D3D9 libraries. It basically was my own (very basic) engine, based on dx9 libraries. It was able to render the map data you passed to it, and edit its heightmaps, 3D objects, and monster spawns.
    • A lot of minor ones.
  • When I stopped working on those tools, I wanted to create a simple game. I started another vb.net + dx9 project, but it was obviously a very complex thing to do, so I started to study/use C#, and learn Unity a little bit. I then dropped it because university.
  • After a while (2014/15) I started to work for a startup company: Crimson Games. We were a very small team, working on a big MMORPG project, Heroes of Asgard, which then got stopped halfway in 2016. I was working on the client, including:
    • Client side scripts
    • Entities sync with the server : players, actions, skills, monsters, items, etc.
    • Movements and inputs
    • Animators for characters and monsters
    • UI : menus, inventory, skills, buffs, minimap system, etc.
    • Map creation : worked on 3 big maps. I designed them created the terrain and vegetation, and then filled with assets our modeller did. I also created a daytime and weather system for our maps, which was then sync'd with the server.
    • Some shaders for multiple effects: rain/snow reacting materials for environment and characters, custom water shader(not the one below), distortions, animated UI materials
    • Basic visual effects : particle systems, other various VFX (example)
  • I worked on Habitat, for IndieVault gamejam 2016. I did the controls/characters movements, and some graphics.
  • I then started some other unity projects as I was experimenting and learning new stuff, planning to release some of them on the asset store too: (in no particular order)
    • AudioSync
      A plugin useful for various effects based on audio signals, for example automatically animating your character as he/she speaks. It performs an FFT on the audioclip samples and provides frequency bands and other data, whit a customizable output function, which you can then use for whatever you want.
    • FloWater
      Started it as a simple water shader while I was learning shader coding, with just refraction and 1 direction flow. After some time I decided to improve it, adding dynamic waterflow and more customizable colors. Eventually I will reflections and other stuff too.
    • BendinGrass
      I liked shaders, so I wanted to try Compute Shaders for grass touch bend. While I was learning about compute shaders, I noticed that it could have been a good package to release too. I had to drop compute shaders for compatibility, and use normal shaders instead. I worked on this a lot, it now provides its own rendering system, specialized in packing volumetric data on a texture 2D and supporting hundreds of objects thanks to the GPU computing.
    • Other minor things I was working on:
      • A triplanar shader pack, with some variants, like heightmap blending, bumpmap, splat maps, ecc.
      • A sign creator.
    • Also did this one for fun: SnoopBoard. A very simple drawing software with the essential drawing functions (draw&erase, undo&redo, generate a Snoop Dogg, etc.) using compute shaders.
